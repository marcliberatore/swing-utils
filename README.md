# swing-utils

Useful functions for interfacing Clojure to Swing.

## Usage

The latest stable release is 0.1.0.

Add this :dependency to your Leiningen project.clj:

["marcliberatore.swing-utils" "0.1.0"]

Feedback and pull requests welcome!
